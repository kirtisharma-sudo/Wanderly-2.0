# Overview

**Wanderly 2.0** is a modern travel booking platform designed to provide a luxurious and immersive browsing experience.

Unlike a traditional frontend project, Wanderly focuses on creating interactions that feel smooth, intuitive, and engaging—from animated page transitions and dynamic destination discovery to persistent user preferences and premium UI micro-interactions.

The goal isn't just to display destinations.

It's to make users **feel like they're already planning their next journey.**
